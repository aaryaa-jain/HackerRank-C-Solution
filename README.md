# HackerRank C Solutions

This repository contains my solutions to selected HackerRank programming problems implemented in the C language. The solutions are organized for clarity and ease of reference.

## Overview
- Platform: HackerRank  
- Language: C  
- Focus: Problem-solving, logic building, and core C concepts

## Structure
Problems are grouped based on topics such as:
- Basics and Input/Output
- Conditional Statements
- Loops
- Arrays and Strings
- Functions and Pointers

## Purpose
The purpose of this repository is to:
- Practice and strengthen C programming skills
- Maintain a personal archive of solved problems
- Track learning progress over time

## Usage
These solutions are intended for educational and reference purposes. Readers are encouraged to attempt the problems independently before reviewing the code.
